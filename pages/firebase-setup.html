How to Request App Access

Important Notes:

  • Only Safari is supported. Other browsers cannot be used for this process.
  • Registration requires a Google account email. Non-Google account emails will not work.
  1. Use the Firebase invite link to register your Google account email for app access. You will receive an email from Firebase.
  2. Complete the setup process as outlined in the official Firebase setup page. Please, note that the app is released with Ad Hoc mode.
  3. Wait for a new build that's compiled with your device UDID included. This process is automated. So, it only takes approximately 7 minutes. The project that automated this process is documented here.
  4. Once you receive an email update from Firebase, follow the Apple official guide to enable Developer Mode.
  5. Download and install the app by following the instruction from the email from Firebase.

The source code of the app can be found here.
